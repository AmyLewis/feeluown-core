feeluown core
=============

feeluown-core 是 `feeluown <https://github.com/cosven/feeluown>`_ 的核心模块。

.. note::

   目前，项目 README 中对此项目的定位为 ``音乐播放服务器`` ，但其实并不是那样子。
   此项目主要是为 feeluown 服务，作为 feeluown 的一个模块存在。
   README 中功能演示，以后也会转移到 feeluown 项目中去。
